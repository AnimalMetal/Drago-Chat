&nbsp;    # NVDA Chat

&nbsp;    

&nbsp;    Accessible chat addon for NVDA screen reader.

&nbsp;    

&nbsp;    ## Features

&nbsp;    - Private messaging

&nbsp;    - Group chats with admin controls

&nbsp;    - Friend management

&nbsp;    - Local message storage

&nbsp;    - Customizable sounds and notifications

&nbsp;    

&nbsp;    ## Installation

&nbsp;    1. Download the latest .nvda-addon file from Releases

&nbsp;    2. Open the file with NVDA running

&nbsp;    3. Follow the installation prompts

&nbsp;    

&nbsp;    ## Version

&nbsp;    Current version: 2.0.0
