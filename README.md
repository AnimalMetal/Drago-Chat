## What is NVDA Chat?

NVDA Chat is a messaging addon for NVDA that lets you:
- Send private messages to friends
- Create and manage group chats
- See who's online
- Save your chat history
- Navigate messages easily with your keyboard

Everything works with your screen reader - no visual interface needed!

---

## Getting Started

### Step 1: Install the Addon

1. Download the `.nvda-addon` file
2. Press Enter on the file
3. NVDA will ask "Do you want to install this addon?"
4. Press Enter to confirm
5. Restart NVDA when prompted

**That's it! The addon is installed.**

---

### Step 2: Open NVDA Chat

You can open NVDA Chat in two ways:

**Method 1: Using Input Gestures**
1. Go to NVDA menu (NVDA+N)
2. Select "Preferences"
3. Select "Input Gestures"
4. Find "NVDA Chat" category
5. Assign a gesture to "Open Chat Window"
6. Press OK

**Method 2: From NVDA Menu** (if configured by administrator)
Some installations may have NVDA Chat in the Tools menu.

---

### Step 3: Create Your Account

The first time you open NVDA Chat, you'll see the Account Setup window.

**Fill in the form:**
- **Username:** Choose a username (this is what others will see)
- **Password:** Create a secure password
- **Email:** Your email address
- Press Tab to move between fields
- Press the "Register" button when done

**Success!** You'll hear "Registration successful!"

The chat window opens automatically.

---

### Step 4: Connect to the Server

After registering, you're automatically connected. But if you get disconnected:

**To connect:**
- Press `Ctrl+C` in the NVDA Chat window
- Or use File menu ‚Üí Connect
- You'll hear "Connected"

**You're ready to chat!**

## Basic Features

### Adding a Friend

Before you can chat with someone, you need to add them as a friend.

**How to add a friend:**

1. In the NVDA Chat window, press `Ctrl+F` (or Friends menu ‚Üí Manage Friends)
2. Type the username you want to add in the text box
3. Press the "Add Friend" button (or press Enter)
4. You'll hear "Friend request sent to [username]"

**Wait for them to accept your request.**

When they accept, you'll hear a notification sound and see them in your chat list!

---

### Sending Your First Message

**To send a private message:**

1. Open NVDA Chat window
2. Your chat list is on the left side
3. Use **Up/Down arrows** to select a friend
4. Press **Enter** to open the chat
5. The chat opens on the right side
6. Type your message in the message input box
7. Press **Enter** to send

**That's it!** Your friend will receive the message.

---

### Receiving Messages

When someone sends you a message:

1. You'll hear a notification sound (ding!)
2. NVDA will say "Message from [username]"
3. The message appears in your chat window if it's open
4. An unread count appears next to their name in the chat list (e.g., "John - 3 unread")

**To read the message:**
1. Select their name in the chat list (use Up/Down arrows)
2. Press Enter to open the chat
3. Their messages appear in the Chat History area

---

### Navigating Messages

**Reading message history:**

There are two ways to read old messages:

**Method 1: Using Arrow Keys (Line by Line)**
1. Press Tab until you reach the "Chat History" area
2. Use **Up/Down arrows** to read line by line
3. Each line is one message with format: "Username; message ; timestamp"

**Method 2: Using Page Up/Down (Message by Message)**
1. You can stay in the message input box (where you type)
2. Press **Page Up** to hear the previous message (older)
3. Press **Page Down** to hear the next message (newer)
4. NVDA reads each complete message as you navigate
5. Press **Shift+Page Up** to jump to the oldest message
6. Press **Shift+Page Down** to jump to the newest message

**Quick navigation is perfect for:**
- Quickly reviewing recent messages
- Finding a specific message
- Catching up on history without leaving the input box

---

### Going Back to Chat List

After reading messages in a chat, you can go back to see all your chats:

**Press the "Back" button** (or press `Alt+Left Arrow`)

You're now back at the chat list. Select another chat and press Enter to open it!

## Intermediate Features

### Creating a Group Chat

Want to chat with multiple people at once? Create a group!

**How to create a group:**

1. In NVDA Chat window, press the "New Chat" button
2. Select "Group Chat" from the dialog
3. Press Enter
4. The Create Group window opens
5. **Group Name:** Type a name for your group (e.g., "Study Group")
6. Press Tab to move to the members list
7. **Select members:**
   - Use Up/Down arrows to browse your friends
   - Press **Space** to check/uncheck a friend
   - You can select multiple friends
8. Press the "Create Group" button
9. You'll hear "Group created successfully!"

**Your group is ready!** Everyone you selected is now in the group.

---

### Sending Group Messages

Group messages work just like private messages:

1. In your chat list, group chats show "(Group)" or "(Group - You are admin)" after the name
2. Select the group with Up/Down arrows
3. Press Enter to open it
4. Type your message
5. Press Enter to send

**Everyone in the group receives your message!**

---

### Viewing Group Members

Want to see who's in a group?

**How to view members:**

1. Select the group in your chat list
2. Press the **Application Key** (usually between right Alt and right Ctrl)
3. Select "View Members"
4. A dialog shows all members
5. The admin is marked with "(Admin)" next to their name
6. Press OK to close

---

### Managing Your Group (Admin Only)

If you created the group, you're the admin. You can manage it!

**To manage your group:**

1. Select your group in the chat list
2. Press the **Application Key**
3. Select "Manage Group (Admin)"

**In the management window, you can:**

**Add Members:**
1. On the left side, you'll see "Available Friends"
2. Select a friend from this list
3. Press "Add Selected Member"
4. They're added to the group instantly!

**Remove Members:**
1. On the right side, you'll see "Current Members"
2. Select a member
3. Press "Remove Selected Member"
4. They're removed from the group

**Transfer Admin Rights:**
1. Select a member from "Current Members"
2. Press "Transfer Admin to Selected"
3. Confirm when asked
4. They become the new admin immediately
5. You become a regular member

Press "Close" when done.

---

### Leaving a Group

Don't want to be in a group anymore?

**How to leave:**

1. Select the group in your chat list
2. Press the **Application Key**
3. Select "Remove from My List"
4. The group disappears from your list

**Note:** The group still exists for others. You just left it.

---

### Deleting a Group (Admin Only)

As admin, you can delete the entire group for everyone.

**How to delete:**

1. Select your group in the chat list
2. Press the **Application Key**
3. Select "Delete Group for Everyone (Admin)"
4. A warning dialog appears: "Delete '[GroupName]' for EVERYONE? This cannot be undone!"
5. Press Yes to confirm
6. The group is deleted for ALL members

**Warning:** This permanently deletes the group. Use with caution!

## Advanced Features

### Muting Chats

Too many notifications from a busy chat? Mute it!

**How to mute a chat:**

1. Select the chat in your list
2. Press the **Application Key**
3. Select "Mute Chat"
4. You'll hear "Chat muted - no sounds or speech notifications"

**What happens when muted:**
- No notification sounds from this chat
- No "Message from..." speech announcements
- Messages still appear in the window when you open it
- Unread count still works
- The chat shows "[Muted]" at the end of its name

**To unmute:**
1. Select the muted chat (it will say "[Muted]")
2. Press the **Application Key**
3. Select "Unmute Chat"
4. You'll hear "Chat unmuted"

---

### Using /me Actions

Want to express an action instead of just talking?

**How to use /me:**

In the message input box, type:
```
/me waves hello
```

Then press Enter.

**Everyone sees:**
```
YourName waves hello
```

Instead of the usual:
```
YourName: waves hello
```

**Perfect for:** roleplay, expressing actions, being playful, showing emotions!

**Examples:**
- `/me laughs` ‚Üí "YourName laughs"
- `/me is thinking` ‚Üí "YourName is thinking"
- `/me left the room` ‚Üí "YourName left the room"

---

### Checking for Updates

Want to make sure you have the latest version?

**How to check:**

1. Open NVDA Chat window
2. Press `Ctrl+S` (or Settings menu ‚Üí Settings)
3. You're in the General tab
4. Tab until you hear "Check for Updates Now" button
5. Press Enter
6. NVDA Chat connects to GitHub to check for updates
7. Wait a moment...

**If you're up to date:**
- You'll hear "You have the latest version (X.X.X)"

**If an update is available:**
1. A dialog shows the new version and changelog
2. Example: "Update Available. Version 2.2.0 is available. Would you like to download and install it?"
3. Press "Yes" to download
4. The update downloads automatically
5. NVDA's installer opens
6. Follow the prompts to install
7. Restart NVDA when done

**Automatic updates:**
By default, NVDA Chat checks for updates when NVDA starts. You can disable this in Settings ‚Üí General tab.

### Saving Messages Locally

Want to keep a permanent record of your chats?

**NVDA Chat automatically saves your messages!**

**Where are they saved?**
By default: `C:\Users\[YourName]\NVDA Chat Messages\`

**File format:**
Each chat has its own text file with messages in this format:
```
username; message ; 2026-02-03 14:30:45
username2; reply ; 2026-02-03 14:31:12
```

**You can:**
- Open these files anytime with Notepad
- Back them up to another location
- Search through them
- Keep them forever

**To change the save location:**
1. Go to Settings (Ctrl+S)
2. General tab
3. Find "Messages Folder"
4. Press "Browse" button
5. Choose a new folder
6. Press OK

**To disable saving:**
Uncheck "Save messages locally" in Settings ‚Üí General

---

### Online/Offline Status

See who's available to chat!

**In your chat list:**
- Private chats show "(online)" or "(offline)" next to names
- **Example:** "John (online)" or "Mary (offline)"
- Online friends are available to chat right now
- Offline friends will see your messages when they come back online

**Your status:**
- You're "online" when connected (press Ctrl+C)
- You're "offline" when disconnected (press Ctrl+D) or when NVDA is closed
- Your friends see your status automatically

**Note:** You can send messages to offline friends. They'll receive them when they connect!

---

### Hiding Timestamps

Don't like seeing timestamps in messages?

**How to hide them:**

1. Go to Settings (Ctrl+S)
2. General tab
3. Find "Show date and time in messages" checkbox
4. Uncheck it
5. Press OK

**Before (with timestamps):**
```
John; Hello there ; 2026-02-03 14:30:45
Mary; Hi John ; 2026-02-03 14:31:12
```

**After (without timestamps):**
```
John; Hello there
Mary; Hi John
```

## Settings

Press `Ctrl+S` in NVDA Chat to open Settings.

### General Tab

**Sound Settings:**
- **Enable sounds:** Master switch - turns all sounds on/off
- **Play sound on message received:** Hear a "ding" for new private messages
- **Play sound on message sent:** Hear a sound when you send a message
- **Play sound on group message:** Hear a sound for group chat messages
- **Play sound on user online:** Hear when a friend comes online
- **Play sound on user offline:** Hear when a friend goes offline

**Reading Messages:**
- **Read messages aloud:** Have NVDA speak incoming messages
- **Read message notifications in group chats:** Hear "John in Study Group: message text"
- **Read message notifications in private chats:** Hear "Message from John: message text"
- **Read aloud when you send a message:** Hear your own sent messages

**Message Storage:**
- **Save messages locally:** Keep a permanent copy of all messages on your computer
- **Messages folder:** Where to save messages (click Browse to change location)

**Display:**
- **Show date and time in messages:** Toggle timestamps on/off in message history

**Updates:**
- **Check for Updates Now** button - Manually check for new versions
- **Check for updates automatically on startup:** Auto-check when NVDA starts

---

### Sounds Tab

Customize each notification sound individually:

Each sound has a checkbox to enable/disable it:
- **Message received sound** - New private message arrives
- **Message sent sound** - You send a message
- **Group message sound** - New group message arrives
- **User online sound** - A friend comes online
- **User offline sound** - A friend goes offline
- **Connected sound** - You connect to the server
- **Disconnected sound** - You disconnect from the server
- **Friend request sound** - Someone sends you a friend request
- **Error sound** - An error occurs

**Note:** The master "Enable sounds" switch in General tab must be checked for these to work.

---

### Account Tab

View and manage your account information:

**Your Information:**
- Username (read-only)
- Email (read-only)
- Server URL (read-only)

**Actions:**
- **Change Password** button - Set a new password for your account
- **Logout** button - Sign out (you'll need to log in again)

## Troubleshooting

### "Cannot connect to server"

**Problem:** NVDA Chat can't reach the server.

**Solutions:**
1. Check your internet connection (open a website to test)
2. Make sure the server is online (ask other users or the administrator)
3. Check if your firewall is blocking NVDA Chat
4. Try disconnecting (`Ctrl+D`) then reconnecting (`Ctrl+C`)
5. Restart NVDA completely

---

### "Username already exists"

**Problem:** Someone already registered that username.

**Solution:** Choose a different username. Usernames must be unique.

---

### "Incorrect password"

**Problem:** The password you entered is wrong.

**Solutions:**
1. Make sure Caps Lock is OFF
2. Re-type your password carefully
3. If you forgot your password, contact the server administrator (they can reset it)

---

### Messages not saving

**Problem:** Your messages aren't being saved to files.

**Solutions:**
1. Open Settings (Ctrl+S) ‚Üí General tab
2. Make sure "Save messages locally" is CHECKED
3. Check the messages folder path exists
4. Make sure Windows can write to that folder (check permissions)
5. Try changing the messages folder to your Documents folder

---

### Sounds not playing

**Problem:** You don't hear notification sounds.

**Solutions:**
1. Settings ‚Üí General ‚Üí Make sure "Enable sounds" is CHECKED
2. Settings ‚Üí Sounds tab ‚Üí Check individual sound checkboxes
3. Make sure your computer volume is up
4. Test NVDA's other sounds to make sure NVDA audio works
5. Restart NVDA

---

### Friend request not going through

**Problem:** Your friend request failed or wasn't received.

**Solutions:**
1. Make sure you typed the username correctly (usernames are case-sensitive)
2. Check that you're connected to the server (you should see "Connected")
3. The user might have already received a request from you
4. Ask them to check their Friends list for pending requests

---

### Can't see new messages with Page Up/Down

**Problem:** Page Up/Down doesn't show the latest messages.

**Solution:** This was a bug in older versions. Make sure you have version 2.2.0 or later. New messages are now automatically added to the history.

---

### Update check fails

**Problem:** Checking for updates doesn't work.

**Solutions:**
1. Check your internet connection
2. Make sure you can access GitHub (some networks block it)
3. Try again in a few minutes
4. If it still fails, download updates manually from the GitHub releases page

---

### Can't navigate to a chat with Enter key

**Problem:** Pressing Enter on a chat doesn't open it.

**Solution:** Make sure you're focused on the chat list (left side). Press Tab or Shift+Tab to move focus if needed.

## Tips and Tricks

### Organizing Your Chats

- Chats appear in order of most recent activity (newest at top)
- Unread messages show a count: "John - 3 unread"
- Mute busy group chats to reduce notification noise
- Delete old chats you don't use (they'll reappear if someone messages you)

### Efficient Messaging

- Use Page Up/Down to quickly review messages without leaving the input box
- Use `/me` for expressive actions and roleplay
- Use Shift+Page Up to jump to the start of a conversation instantly
- Mute chats during work to focus, unmute when you want to chat

### Privacy and Safety

- Your messages are saved only on YOUR computer (not on the server)
- Only people you add as friends can message you
- You can leave any group chat at any time
- Block unwanted contacts by removing them from your friends list
- Your password is encrypted and never shown in plain text

### Group Chat Management

- Give groups clear, descriptive names ("Work Team", "Study Group", etc.)
- Only add people who want to be in the group
- Transfer admin to someone active if you're leaving or inactive
- Delete groups you no longer need to keep your list clean

### Keyboard Efficiency

- Learn the shortcuts: Ctrl+C (connect), Ctrl+D (disconnect), Ctrl+F (friends), Ctrl+S (settings)
- Use Application Key for quick actions (mute, view members, delete)
- Use Page Up/Down instead of arrow keys for faster message navigation
- Use Shift+Page Up/Down to jump to ends of conversations

## Frequently Asked Questions (FAQ)

**Q: How do I open NVDA Chat?**
A: Assign a gesture in Input Gestures (NVDA menu ‚Üí Preferences ‚Üí Input Gestures ‚Üí NVDA Chat category), or check if your administrator added it to the Tools menu.

**Q: Can I use NVDA Chat offline?**
A: No, you must be connected to the server to send and receive messages. However, you can read your saved message history offline.

**Q: How many friends can I add?**
A: There's no limit! Add as many friends as you want.

**Q: How many people can be in a group?**
A: There's no specific limit, but very large groups (50+ people) might be harder to manage.

**Q: Can I change my username?**
A: No, usernames are permanent. You would need to create a new account with a different username.


**Q: Can I send files or images?**
A: Not in the current version. This may be added in a future update.

**Q: Does NVDA Chat work with NVDA 2024.x?**
A: NVDA Chat is tested with NVDA 2023.1 and later. It should work with 2024.x versions.

**Q: Is there a mobile app?**
A: No, NVDA Chat only works on Windows with NVDA screen reader.

**Q: Can I export my message history?**
A: Yes! Your messages are saved as plain text files. Just copy them from the messages folder.

**Q: What happens if I lose my password?**
A: Contact your server administrator. They can reset your password.
